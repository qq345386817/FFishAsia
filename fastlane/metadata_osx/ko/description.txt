Mac용 Little Nature는 아시아 동물과 식물의 CC0 기반 3D 생물 모델 기록을 탐색하기 위한 데스크톱 도감입니다. 더 넓은 작업 공간에서 카탈로그를 살펴보고 모델 상세 정보를 확인하며 저장된 모델 콘텐츠를 관리할 수 있습니다.

자연사 참고, 교육, 모델 선택 또는 꽃, 곤충, 양서류, 해양 생물 등 다양한 생명체의 형태와 분류를 둘러보는 데 사용할 수 있습니다.

주요 기능:
- 데스크톱 카탈로그: Mac에서 엄선된 3D 동물 및 식물 모델 기록 탐색
- 카테고리 탐색: 전체 / 동물 / 식물 / 특별 필터 지원
- 빠른 검색: 일본어, 영어, 중국어 또는 학명으로 모델 찾기
- 필요할 때 다운로드: 저장 공간 사용을 줄이기 위해 필요한 모델만 다운로드
- 오프라인 접근: 다운로드한 모델 콘텐츠 다시 보기
- 다운로드 관리: 저장된 모델 확인 및 로컬 콘텐츠 삭제
- 모델 상세 정보: 면 수, 정점 수, 파일 크기, 카테고리, 출처 링크 보기

모델 리소스는 공개 출처에서 정리되었으며 CC0 라이선스 조건에 따라 탐색과 로컬 참고용으로 사용됩니다. Little Nature는 계정 기능을 제공하거나 개인 데이터를 업로드하지 않습니다.
